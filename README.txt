The IronJacamar project
=======================

This is the home of the IronJacamar project which implements the 

 * Java Connector Architecture 1.6 (JSR-322)

standard as well as additional resource adapters needed by 
JBoss Application Server.

The documentation for the project is located here:

  http://www.jboss.org/ironjacamar/docs

Please, visit our forums for questions or design discussions

  User:      http://community.jboss.org/en/ironjacamar?view=all
  Developer: http://community.jboss.org/en/ironjacamar/dev?view=all

For more information about the project visit our homepage:

  http://www.jboss.org/ironjacamar
